Detailed Description

Represents an observable 3D object.

Contains a model, a set of uniform collections, a shader, animations, a skeleton.

Warning
I think this class contains a bit too much implementation. (see draw method). It seems weird that "entity" is responsible for binding and clearing all uniform data for the shader.
